Introduction

What can I find on this site?

This site is intended to help you design your own e-portfolio directly or indirectly in HTML. We provide you with very useful applications, development tips and a complete bibliography so that you have all the tools at your fingertips. Our goal is not to restrict you but to help you build an e-portfolio that suits you.

How would I design my own e-portfolio in HTML ?

As we said above, we propose you to design your own e-portfolio in HTML, directly or indirectly. Contrary to the previous years, we decided not to expose only Wix (to see the reasons), we also propose the use of the web hosting service offered by Github. There, you can upload your web pages (HTML) that you can build via Seamonkey or directly in HTML with a source code editor (like VS Code, github.dev, Notepad,...).

Which tool should I use ?

In this tuto, we will expose two applications : SeaMonkey and VS Code. SeaMonkey, because it is a free and open-source Internet suite. VS Code, because it is practical and already installed on the N7's computers.

Beware

Whatever the source code editor is, the parts about "Code in HTML" and "Complements in HTML" (except for the VS Code installation) works.
