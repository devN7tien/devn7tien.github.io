Use GitHub

Why GitHub ?

GitHub is a very useful sharing and storage platform. It can be a great help for your projects. GitHub also offers the possibility to store your web pages and thus to show your e-portfolio.

How to install GitHub ?

First Step : create an account

Go to GitHub and click on Sign Up

Beware

Your website's URL (i.e. e-portfolio) will be built with your username: username.github.io

Second Step

Start a new repository and make sure you complete the correct procedure as indicated.

Third Step

Go on Code, upload all your web pages on your repository (you can simply drag the files into the repository) and commit.

Fourth and Last Step

Check that your website is correctly implemented : go on Settings then on Pages, you'll normally see a sentence equivalent to the one underlined.
